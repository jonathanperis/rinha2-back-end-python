Repo-wide community health files are centralized in `jonathanperis/.github` (100% community health score).

Files hosted there: CODE_OF_CONDUCT, CONTRIBUTING, SECURITY, Pull Request templates.

**How to apply:** Never create SECURITY.md, CODE_OF_CONDUCT.md, CONTRIBUTING.md, or PR templates in individual repositories — they inherit from the org-level `.github` repo automatically.
